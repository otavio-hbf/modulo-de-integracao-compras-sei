# modulo-de-integracao-compras-sei (MICS-IFPE)
Projeto da disciplina de IF1008 - Engenharia de Requisitos 2026.1 com o intuito de criar um módulo integração entre o compras.net e o SEI! para facilitar, para o Instituto Federal de Pernambuco (IFPE), o fluxo de cadastro de artefatos em ambas plataformas.
